## ADDED Requirements

### Requirement: Establish Target Budgets
The system SHALL allow the user to establish a target budget amount for specific expense categories.

#### Scenario: User sets a budget for a category
- **WHEN** the user navigates to the Budgets section
- **THEN** the user can input a numerical target amount for a category (e.g., "Alimentation: 400€/month").

### Requirement: Track Budget Progress
The system SHALL visually track the progress of current spending against the established target budgets.

#### Scenario: User checks budget status
- **WHEN** the user views the dashboard or budget view
- **THEN** the system displays a progress bar or percentage indicating how much of the budget has been consumed for the current month.
